# Readme: Ecophysiology of two mesophotic octocorals code and data
Publicly available code utilized in analyses for manuscript 'Ecophysiology of two mesophotic octocorals intended for restoration: Effects of light and temperature' (Lange et al, 2025). Data in analyses is included as .csv files in 'data' folder. R code is broken up by thermal tolerance and photobiology in the 'analysis' folder.

Publication can be found here: https://doi.org/10.1002/lno.70214
